Bienvenido al Sistema preventivo

Esta aplicacion fue realizada con el objetivo de tener un mecanismo para lidiar en contra de los delincuentes que realizan llamadas telefónicas para realizar estafas y fraudes. Los usuarios de la aplicación pueden ayudarse mutuamente para combatir este tipo de delitos trabajando colaborativamente.

Hola!, las siguientes son las funciones que puedes realizar en esta aplicación.

Consultar número telefónico

Para poder hacer uso de esta funcionalidad debe dirigirse al menú que se encuentra
en la parte lateral izquierda de la aplicación y dar click a la opción "Consultar número telefónico"
Luego, se le solicitará digitar el número telefónico que desea consultar, el sistema automáticamente
le hará saber reportes realizados por otros usuarios sobre este o, en otro caso, informarle
que no existen reportes asociados a dicho número.

Realizar reporte

Para poder hacer uso de esta funcionalidad debe dirigirse al menú que se encuentra
en la parte lateral izquierda de la aplicación y dar click a la opción "Realizar reporte"
Luego, podrá acceder a un formulario donde podrá realizar su reporte asociado con un
número telefónico que usted quiera reportar.
